Beim synchronen Veröffentlichungsmodus wird nach erfolgreichem Hochladen der BOM auf die Analyseergebnisse gewartet.

Damit können die Analyseergebnisse direkt in Jenkins pro Build gespeichert und dargestellt werden - inklusive interaktivem Verlaufsdiagram.

Der synchrone Veröffentlichungsmodus ist mit Dependency-Track Version 3.3.1 und neuer möglich.
